## Web Crawling VS Web Scraping
- Web Crawling: 웹페이지 내에서 허용된 링크를 따라 가며 데이터를 마구잡이로 가져오는 것
- Web Scraping: 웹페이지 내에서 필요한 부분의 데이터만 가져오는 것
